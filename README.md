# Online Room Reservation System

## Project Description
This project is a Java web-based Online Room Reservation System developed for Ocean View Resort. It automates hotel reservation management using JSP, Servlets, and MySQL.

## Features
- User login and session management
- Add reservation
- View reservations
- Edit reservation
- Delete reservation
- Automatic bill calculation
- Print bill
- Help page
- JUnit automated testing

## Technologies Used
- Java
- JSP
- Servlets
- MySQL
- Apache Tomcat
- Bootstrap
- JUnit

## Database Setup
1. Create a MySQL database named `hotel_db`
2. Import the SQL script from `database/hotel_db.sql`

## How to Run
1. Open the project in NetBeans
2. Configure Apache Tomcat
3. Configure MySQL database connection
4. Run the project
5. Access through:
   `http://localhost:8080/RoomReservationSystem`
